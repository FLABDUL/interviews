## About

This repository serves as a collection of problems typically asked in coding interviews. The problems include various topics such as algorithms, data structures, system design, and more. Each problem is designed to challenge and improve your problem-solving skills, with the goal of preparing you for software engineering interviews at top companies.

The repository is organized in a way that makes it easy to explore different problem types, and each problem comes with its own solution implementation and test cases.
